# kaholo-plugin-gsuite
G-Suite plugin for Kaholo

## Method: Create User

### Parameters
1. Credentials: (Vault)
2. Super admin email
3. Family name
4. Given Name
5. Password
6. Change password at next login (boolean)
7. Is suspended (boolean)
8. Primary Email
9. Add user to groups

## Method: Add a user to group

### Parameters
1. Credentials: (Vault)
2. Super admin email
3. Group name
4. Primary Email
